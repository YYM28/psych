# Как редактировать сайт

Этот документ объясняет, как менять тексты на сайте после того, как он опубликован на GitHub Pages. Не нужно знать HTML — нужно понимать только пару простых правил.

## Где лежат тексты

Сайт состоит из трёх HTML-файлов:

- `index.html` — главная страница
- `about.html` — страница «Подробнее обо мне»
- `education.html` — страница «Подробнее об образовании»

Каждый файл это просто длинный текстовый документ с HTML-разметкой. Тексты лежат внутри тегов вида `<p>...</p>` или `<h1>...</h1>`.

## Главное правило

**Менять можно только текст, не сами теги.**

Например, если в файле написано:

```html
<h1>Найти опору, когда всё кажется запутанным</h1>
```

Менять можно текст `Найти опору, когда всё кажется запутанным`.
Не трогать `<h1>` и `</h1>`.

После правки получится, например:

```html
<h1>Новый заголовок</h1>
```

## Как редактировать через GitHub в браузере

1. Зайди на свой репозиторий на GitHub.
2. Открой файл (например, `index.html`).
3. Нажми на иконку карандаша «Edit this file» в правом верхнем углу.
4. Найди текст, который хочешь поменять (используй Ctrl+F / Cmd+F).
5. Поменяй текст.
6. Прокрути вниз до «Commit changes».
7. Напиши короткое описание изменения, например «Изменила цену сессии».
8. Нажми «Commit changes».

Через 30–60 секунд изменение появится на сайте.

## Структура файлов — где что искать

В файле `index.html` блоки помечены большими комментариями для удобной навигации. Используй Ctrl+F и ищи по этим меткам:

```
<!-- HERO -->                       — главный экран
<!-- БЛОК "КОГДА ЧТО-ТО МЕНЯЕТСЯ" -->  — текст про переходы
<!-- БЛОК "С ЧЕМ ПРИХОДЯТ" -->         — 5 портретов
<!-- БЛОК "ОБО МНЕ" -->                — короткое обо мне
<!-- БЛОК "КАК ПРОХОДИТ РАБОТА" -->   — метод
<!-- БЛОК "ОБРАЗОВАНИЕ" -->            — короткий список образования
<!-- БЛОК "ФОРМАТ И УСЛОВИЯ" -->       — длительность, частота, формат
<!-- БЛОК FAQ -->                      — частые вопросы
<!-- ФИНАЛ + CTA -->                   — финальный блок и Telegram
```

## Конкретные примеры частых правок

### Поменять цену сессии

В файле `index.html` найди блок «ФОРМАТ И УСЛОВИЯ». Если решишь добавить цену, можешь использовать структуру одного из существующих столбцов как образец.

### Поменять Telegram

В файле `index.html` найди по поиску `t.me/msyncing`. Замени везде `msyncing` на новый ник.
То же самое в `about.html` и `education.html`.

### Обновить статус обучения

После того как завершишь программу психоаналитической психотерапии в 2026, найди в `index.html` фразу «завершение 2026» и замени на год получения сертификата. То же в `education.html`.

### Поменять FAQ

В `index.html` найди блок FAQ. Каждый вопрос обёрнут в `<details>...</details>`. Внутри `<summary>` лежит вопрос, внутри `<div class="faq-answer">` лежит ответ из нескольких `<p>...</p>`.

Чтобы добавить новый вопрос, скопируй один из существующих блоков `<details>` целиком и измени текст.

### Добавить новую сертификацию

В `index.html` найди блок «ОБРАЗОВАНИЕ». Список лежит в одном большом `<p>`, элементы разделены `<br>`. Просто добавь новую строку и `<br>` после неё.

Также продублируй на странице `education.html` — там подробный список с датами.

## Что делать, если что-то поломалось

Если после правки сайт начал выглядеть странно:

1. Зайди в раздел «Commits» репозитория.
2. Найди свой последний коммит.
3. Открой его и нажми «Revert».
4. Это откатит изменения. Сайт вернётся к рабочему состоянию.

После этого можно попробовать снова, более аккуратно.

## Что менять можно без страха

- Любой текст между `<p>` и `</p>`
- Любой текст между `<h1>` и `</h1>`, `<h2>` и `</h2>`, `<h4>` и `</h4>`
- Любой текст между `<summary>` и `</summary>`
- Любой текст между `<div class="что-то">` и `</div>`, если внутри только текст

## Что НЕ менять

- Сами теги `<...>` и `</...>`
- Атрибуты вроде `class="..."` или `style="..."`
- Содержимое внутри `<style>...</style>` (это CSS, отвечает за оформление)
- Файлы фотографий в папке `photos/` (если хочешь поменять фото — загрузи новое в эту папку и поменяй имя файла в `<img src="photos/имя.jpg">`)

## Если нужна помощь

Любую правку можно сделать через ChatGPT или Claude: открой нужный HTML-файл, скопируй кусок, в котором хочешь что-то поменять, попроси «помоги поменять текст в этом блоке на такой-то». ИИ-помощники хорошо справляются с такими задачами и не сломают разметку.
