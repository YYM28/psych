## Как опубликовать на GitHub Pages

### Шаг 1. Создать репозиторий

1. Зайти на github.com, создать новый репозиторий.
2. Назвать его, например, `marina-site`. Сделать публичным.
3. Загрузить все файлы из этой папки в репозиторий (через веб-интерфейс «Add file → Upload files» или через git).

### Шаг 2. Включить GitHub Pages

1. В репозитории зайти в Settings.
2. Слева — раздел Pages.
3. В разделе «Source» выбрать ветку `main` и папку `/ (root)`.
4. Нажать Save.
5. Через 1-2 минуты сайт будет доступен по адресу `https://username.github.io/marina-site/` (где `username` — твой логин на GitHub).

### Шаг 3 (опционально). Подключить свой домен

Если есть домен, например `marinasushentseva.com`:

1. У регистратора домена создать DNS-записи типа A с указанием на IP-адреса GitHub Pages:
   - 185.199.108.153
   - 185.199.109.153
   - 185.199.110.153
   - 185.199.111.153
2. Создать также CNAME-запись для www: `username.github.io`.
3. В Settings → Pages в поле «Custom domain» написать домен.
4. Поставить галочку «Enforce HTTPS».

DNS обновляется до 24 часов.

## Аналитика (опционально)

Сайт подготовлен под подключение **Plausible Analytics** (privacy-friendly, без cookies, без баннеров согласия). Стоимость от 9$/мес.

### Как подключить:

1. Зарегистрироваться на plausible.io.
2. Добавить свой сайт.
3. В каждом из трёх HTML-файлов (`index.html`, `about.html`, `education.html`) найти комментарий `<!-- АНАЛИТИКА -->`.
4. Раскомментировать строку с скриптом (убрать `<!--` и `-->`).
5. Заменить `ВАШ-ДОМЕН.com` на свой реальный домен.

Альтернатива — Google Analytics 4. Бесплатно, но потребуется баннер согласия cookies (GDPR).

## Что входит в дизайн-систему

**Цвета:**
- `#FAFAF7` — основной фон (тёплый белый)
- `#1F2421` — основной текст (графит)
- `#4A4F4B` — вторичный текст
- `#1F7A8C` — акцент (сине-бирюзовый)
- `#0F3D4A` — тёмный акцент
- `#E8EFF2` — голубая дымка (фон секций)

**Шрифты:**
- Cormorant Garamond — заголовки и основной текст (через Google Fonts)
- Inter — служебный текст, кнопки, метки (через Google Fonts)
